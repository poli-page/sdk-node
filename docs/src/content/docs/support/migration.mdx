This page tracks breaking changes between major versions of `@poli-page/sdk`. For the full per-release diff, see the [changelog](../changelog/).

## v1.0

The first stable release. The full v1.0 surface and the renames from the pre-1.0 line (`client.render(input)` → `client.render.pdf(input)`, etc.) are documented in [`MIGRATION.md` in the repo](https://github.com/poli-page/sdk-node/blob/main/MIGRATION.md).

Short version: every render call now goes through a namespace (`render.pdf`, `render.preview`, `render.document`), all PDF-producing methods require project mode, and inline mode is supported only on `render.preview`.
