Every failure thrown by the SDK is an instance of `PoliPageError`. The `code` field tells you what went wrong; the `requestId` field is what to quote when filing a support ticket.

## The error class

```ts
import { PoliPageError } from '@poli-page/sdk';

try {
  await client.render.pdf({ project: 'billing', template: 'invoice', data: {} });
} catch (err) {
  if (err instanceof PoliPageError) {
    console.error(err.code, err.requestId, err.message);
  } else {
    throw err;
  }
}
```

## SDK-internal codes

These originate inside the SDK before any HTTP call (or in response to a network-layer failure). They are lowercase.

## API codes

Returned by the Poli Page API. Uppercase, snake-case. The SDK forwards the code verbatim.

<ErrorTable errors={[
  { code: 'INVALID_API_KEY', when: 'The API key is malformed or revoked.', recovery: 'Rotate the key in the dashboard.' },
  { code: 'MISSING_API_KEY', when: 'No API key was provided in the request.', recovery: 'Check the constructor was called with `apiKey`.' },
  { code: 'NOT_FOUND', when: 'The `project/template` slug or `documentId` does not exist.', recovery: 'Verify the slug/id; check the template is published in this environment.' },
  { code: 'VALIDATION_ERROR', when: 'The `data` object does not satisfy the template schema.', recovery: 'Inspect the template schema in the dashboard.' },
  { code: 'MISSING_DATA', when: 'The request body lacks a required `data` field.', recovery: 'Pass `data: {}` (or with values) on every render call.' },
  { code: 'QUOTA_EXCEEDED', when: 'You exceeded the per-key rate limit or monthly quota.', recovery: 'Automatically retried; bump tier for sustained higher throughput.' },
  { code: 'PAYMENT_REQUIRED', when: 'The organization`s billing is past due.', recovery: 'Update the payment method in the dashboard.' },
  { code: 'INTERNAL_ERROR', when: 'The API returned 5xx.', recovery: 'Automatically retried; contact support if it persists.' },
]} />

The full list of API codes lives in [the reference](../../reference/errors/).

## Example

```ts
import { PoliPage, PoliPageError } from '@poli-page/sdk';

const client = new PoliPage({ apiKey: process.env.POLI_PAGE_API_KEY! });

try {
  const pdf = await client.render.pdf({
    project: 'billing',
    template: 'invoice',
    data: { invoiceNumber: 'INV-001' },
  });
  // ...
} catch (err) {
  if (err instanceof PoliPageError && err.code === 'QUOTA_EXCEEDED') {
    // Custom strategy beyond the SDK's built-in retries.
    return;
  }
  throw err;
}
```
